## File server

```sh
caddy file-server
	[--root <path>]
	[--listen <addr>]
	[--domain <example.com>]
	[--browse]
	[--templates]
	[--access-log]

docker run --network host -v $PWD:/srv caddy caddy file-server --root /srv --listen localhost:2333 -browse
```
