## Define Commands

```tex
\newcommand{\CmdName}{content}

\newcommand{\bb}[1]{\mathbb{#1}} % command with 1 argument, #1 -- #9 refer to the arguments
\bb{C}

\newcommand{\plusbinomial}[3][2]{(#2 + #3)^#1} % 2 is the default value for the first argument
```
